📱 About the APK

Play API is a powerful media streaming platform that allows users to:

  • 🚀 Search & play multiple songs
  • 🎥 Stream videos without buffering
  • 🔊 Download music & videos instantly
  • 📜 Access lyrics with open-source APIs
  • ⚡ Change playback speed & fast-forward

🌐 Deployment Instructions

GitHub Pages

  1. Upload files to a public GitHub repository.
  2. Go to Settings → Pages and enable GitHub Pages.
  3. Select the main branch and save.
  4. Your site will be live at: https://yourusername.github.io/your-repo/.

Vercel

  1. Sign up at **Vercel**.
  2. Import your **GitHub repository**.
  3. Deploy with default settings (**Static Site**).
  4. Get your **live portfolio URL**!
